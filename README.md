# Google Keep Clone

Pixel-perfect Google Keep UI built with vanilla HTML, CSS & JavaScript.

## How to Run
1. Download or clone this repo
2. Open `index.html` in Chrome, Edge, or any browser
3. No installation needed - it runs offline

## Features
- Responsive
- Note input and grid layout
- Search bar
- Uses Material Icons
